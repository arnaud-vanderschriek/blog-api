# blog-api
# 📖 Blog API (Node.js + Express + MongoDB)

Une API REST simple pour gérer des articles de blog, développée avec **Node.js**, **Express**, et **MongoDB**.

---

## 🚀 Fonctionnalités
- ➕ Créer un post
- 📄 Récupérer tous les posts
- 🔍 Récupérer un post par ID
- ✏️ Mettre à jour un post
- ❌ Supprimer un post
